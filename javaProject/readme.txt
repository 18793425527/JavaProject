estore
	基于servlet和jsp技术的电子商务网站系统。

后台使用主要技术:
	servlet + jsp + mybatis + oracle/mysql
	
该系统包括
	会员信息管理，
	商品管理，
	商品查询，
	购物车管理，
	订单管理等功能模块。

系统需求分析:
	1,用户注册
	2,用户登录
	3,查看商品,列表查询
	4,商品详细页面
	5,将商品添加到购物车
	6,查看购物车
	7,修改购物车
	8,生成订单
	9,结算


环境监测系统介绍

　　　　1.1　　配置模块
　　　　1.2　　日志模块
　　　　1.3　　采集模块
　　　　1.4　　备份模块
　　　　1.5　　网络模块
　　2.服务器　　 接收数据
　　　　1.1　　配置模块
　　　　1.2　　日志模块
　　　　1.3　　入库模块
　　　　1.4　　备份模块
　　　　1.5　　网络模块
	    1.6　　管理员登录模块
